# Brand Walkthrough · Thiocyn

> Die HSB-Mutter-Brand. Jeder im Team wird über die Thiocyn GmbH angestellt. Operativ ist Thiocyn die D2C-Brand mit dem höchsten Volumen im Portfolio.

| Field | Value |
|---|---|
| **Kategorie** | Hair Care · Haarwuchs |
| **Kernbotschaft** | „Volleres Haar? Starten Sie jetzt." |
| **Positionierung** | Wissenschaftlich fundiert, natürliche Wirkstoffe, Made in Germany |
| **Wirkstoff** | Thiocyanat (100% natürlicher körpereigener Wirkstoff) |
| **Zielgruppe** | Männer + Frauen mit Haarausfall (separate Produktlinien) |
| **Garantie** | 100-Tage-Geld-zurück (zentraler Risk-Reversal-Hook in Ads) |
| **Versand** | Kostenlos ab €60 |
| **Schwestermarke** | NOREIZ (Kopfhautgesundheit) |
| **Hauptmarkt** | Deutschland · Sprache Deutsch |
| **Vertriebskanäle** | D2C Shop · Apotheken · Amazon |
| **Website** | [thiocyn.com](https://thiocyn.com) |

## Voice
Sachlich, wissenschaftlich, vertrauensvoll. Versprechen werden mit Wirkmechanismus + Garantie unterlegt — nicht mit Hype.
