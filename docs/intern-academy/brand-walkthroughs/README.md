# Brand Walkthroughs

Kurze 1-Pager-Vorstellungen pro Brand für Fellows. Jede Brand bekommt: Snapshot, Voice, was du als Fellow wahrscheinlich machst, Pointer auf die Tiefen-Quelle (Brand Bible).

> Diese Walkthroughs sind **Kennenlernen, kein operatives Onboarding**. Alle Hires laufen über die Thiocyn GmbH — universelles Hire-Setup steht in [`../thiocyn-employment-onboarding.md`](../thiocyn-employment-onboarding.md).

## Format

Jeder Walkthrough folgt der gleichen Struktur:
1. **Tagline + Snapshot** (1-Liner pro Field-Tabelle)
2. **Voice** — Tone, Stil, Anti-Patterns
3. **Was du als Fellow wahrscheinlich machst** — typische Aufgaben/Projekte
4. **Tiefer einsteigen** — Pointer auf Brand-Bible in Memory + relevante Notion/Repo-Refs

Wenn eine Brand neu dazukommt: kopiere `take-a-shot.md` als Template, passe Felder an.
