# Intern Offboarding Checklist — Hartlimes GmbH

> Owner: Tom Roelants (Intern Lead)
> Scope: Remote Internship Program — all brands
> Last updated: 2026-03-25

---

## Phase 1 — Pre-Offboarding (3–5 days before last day)

- [ ] Confirm last working day with intern and document in Notion
- [ ] Notify Luis (Founders Associate) of offboarding date
- [ ] Request data handover: ask intern to finalize all open Notion notes and Drive files
- [ ] Confirm any open tickets (CS) or open content tasks (Marketing) are reassigned
- [ ] Inform intern of exit interview option (optional, 15 min call)

---

## Phase 2 — Day of Last Day

### Access Revocation

**All Interns**
- [ ] Notion: remove intern from all shared workspaces and pages
- [ ] Google Drive: revoke access to shared brand folders (intern's own subfolder: archive, do not delete)
- [ ] Trello / Asana: deactivate account or remove from all boards/projects

**CS Interns only**
- [ ] Shopify: revoke staff account (Settings → Users → Remove)

**Marketing Interns only**
- [ ] Meta Business Suite: remove user from all connected ad accounts and pages
- [ ] CapCut: remove from shared team workspace (if applicable)

**Confirm NOT applicable (no access was granted)**
- [ ] PayPal — N/A
- [ ] Make (automation) — N/A
- [ ] Supabase — N/A
- [ ] Higgsfield — N/A
- [ ] GetMyInvoices — N/A

### Supabase Intern Portal (Luis or Dev to execute)
- [ ] Set `intern_accounts.is_active = false` for intern's DB record
- [ ] Remove Auth user from Supabase project (`Authentication → Users → Delete`)

### Data Handover
- [ ] Intern confirms all Notion pages are up to date and handed over
- [ ] Drive folder archived under `_offboarded/[Name]_[YYYY-MM]`
- [ ] Open tickets (CS) fully reassigned and documented
- [ ] Open content tasks (Marketing) handed over with status notes

---

## Phase 3 — Post-Offboarding (within 5 business days)

- [ ] Performance note written (brief, internal only — 3–5 sentences max, optional)
- [ ] Exit interview completed or declined (note outcome)
- [ ] Reference letter: if requested → forward to Luis for review and sign-off
- [ ] Equipment return: N/A — fully remote program
- [ ] Offboarding logged in Notion intern tracker (name, last day, status: offboarded)

---

## Notes

- Drive subfolder of intern is **archived, not deleted** — retain for 6 months minimum
- Reference letters are issued on request only, signed by Luis or Peter Hart (MD)
- Any access not listed above (e.g., temporary tool access granted ad hoc) must be manually verified with Luis before offboarding is marked complete
